Welcome to the **GH-900** course sample repository. This repository contains demo materials, exercises, and reference code used throughout the GH-900 course. It is intended to help learners follow along with hands-on labs and understand key concepts covered in the course.

GH-900 is a course designed to introduce learners to GitHub's core features and best practices, including repository management, collaboration workflows, GitHub Actions, and more. This sample repository serves as a companion resource to the course content.
